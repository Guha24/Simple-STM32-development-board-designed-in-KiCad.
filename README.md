# STM32 Development Board PCB Design

This project is a simple STM32-based development board designed using KiCad for learning and practice purposes. The schematic was inspired by Phil’s Lab STM32 hardware design tutorial, while the PCB component placement and routing were implemented independently to gain practical understanding of PCB design workflows.

## Learning Objectives

* Understanding STM32 hardware design basics
* Practicing schematic capture and PCB routing
* Learning PCB layout considerations such as grounding, trace routing, and component placement
* Exploring embedded hardware development workflow

## Notes

This project was created for educational purposes to improve PCB design.
